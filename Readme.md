# 📊 Instagram Follow Checker

Sebuah tool sederhana berbasis Python untuk membandingkan data followers dan following dari hasil export data Instagram dalam format JSON.

---

## ✨ Tentang Project

Project ini membantu kamu untuk menganalisis hubungan follow di Instagram dengan cara membaca data export resmi dari :contentReference[oaicite:0]{index=0}.

Dengan tool ini, kamu bisa mengetahui:

- ❌ Pengguna yang kamu follow tetapi tidak follow balik  
- 🤝 Pengguna yang saling follow (mutual followers)

---

## 🎯 Fitur

- 📥 Membaca data JSON hasil export Instagram  
- 🔍 Mendeteksi akun yang tidak follow balik  
- 🤝 Menampilkan mutual followers  
- ⚡ Proses cepat tanpa API dan tanpa login tambahan  

---

## 📂 Cara Penggunaan

1. Download data Instagram dalam format JSON  
   (Settings → Accounts Center → Download your information → JSON)

2. Masukkan file ke dalam folder project:
   - followers_1.json
   - following.json

3. Jalankan program:
```bash
python app.py
